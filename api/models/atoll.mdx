## Overview

The Atoll model represents the administrative atolls of the Maldives. Atolls are used to group islands for organizational and routing purposes in the shipping system. This model is defined in `app/Models/atoll.php` and uses the `atolls` database table.

<Note>
The Maldives consists of 26 natural atolls administratively grouped into 20 administrative divisions. NaaleMV uses this atoll structure to organize island destinations.
</Note>

## Usage Examples

### Auto-Creating Atolls

From `ShipmentController.php:53-75`:

```php
// Automatically create atoll if it doesn't exist when adding an island
if (!atoll::where('name', strtoupper($request->atoll))->exists()) {
    $atoll = new atoll([
        'name' => strtoupper($request->atoll),
        'code' => '-',
    ]);
    $atoll->save();
}

// Retrieve the atoll for island creation
$atoll = atoll::where('name', $request->atoll)->first();
```

<Note>
When creating islands through the dashboard, atolls are automatically created if they don't already exist. The code field defaults to '-' and can be updated later.
</Note>

### Querying Atolls

```php
// Find atoll by name
$atoll = atoll::where('name', 'MALE')->first();

// Get all atolls
$allAtolls = atoll::all();
```

## Data Management

### Adding New Atolls

Atolls are typically added automatically when islands are created through the admin dashboard (`/dashboard`). You can also manually insert atoll records:

```php
atoll::create([
    'name' => 'HDH',
    'code' => 'HDH',
])->save();
```

### Updating Atoll Codes

If an atoll was created with a placeholder code ('-'), you can update it:

```php
$atoll = atoll::where('name', 'MALE')->first();
$atoll->update(['code' => 'K']);
```

## Integration with Shipping

Atolls are used in the shipping workflow to:

1. **Organize destinations**: Group islands by atoll for route planning
2. **Package routing**: Determine shipping routes based on destination atoll
3. **Address validation**: Verify destination islands belong to valid atolls
4. **Reporting**: Generate reports grouped by atoll
