      Get started in minutes
    </h2>
    <p className="text-base text-gray-600 dark:text-gray-400">
      Follow these steps to set up NaaleMV and start managing your shipping operations
    </p>
  </div>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install the required PHP and JavaScript dependencies using Composer and npm.
      
      ```bash
      git clone https://github.com/RILBERRY/naalemv.git
      cd naalemv
      composer install
      npm install
      ```
    </Step>
    
    <Step title="Configure environment">
      Copy the `.env.example` file to `.env` and configure your database connection and mail settings.
      
      ```bash
      cp .env.example .env
      php artisan key:generate
      ```
      
      Update database credentials in `.env`:
      ```ini
      DB_CONNECTION=mysql
      DB_HOST=127.0.0.1
      DB_PORT=3306
      DB_DATABASE=naalemvdb
      DB_USERNAME=root
      DB_PASSWORD=your_password
      ```
    </Step>
    
    <Step title="Set up the database">
      Run migrations to create all required database tables for shipments, packages, customers, vessels, and islands.
      
      ```bash
      php artisan migrate
      ```
      
      The system will create tables for managing customers, packages, shipments, vessels, islands, atolls, schedules, and receivables.
    </Step>
    
    <Step title="Start the application">
      Compile assets and start the Laravel development server to access the platform.
      
      ```bash
      npm run dev
      php artisan serve
      ```
      
      Access the application at `http://localhost:8000` and log in with your admin credentials.
    </Step>
  </Steps>
</div>
